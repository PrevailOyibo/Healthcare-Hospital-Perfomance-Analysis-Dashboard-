
# Healthcare Hospital Perfomance analysis Project


##  Problem Statement
A Healthcare hospital performance dashboard is a visual representation of data developed to analyse hospital performance. The dashboard enables healthcare managers to monitor patient flow, doctor performance, departmental revenue, waiting time, and treatment outcomes. The dashboard facilitates informed decision making and continous improvement efforts on performance

## Business Problem
Hospital management requires a comprehensive reporting system to monitor operational performance, identify inefficiencies, and make data-driven decisions that improve service delivery and patient satisfaction.

## Project Objectives
The dashboard was developed to answer the following business questions:

Which department treats the highest number of patients?

Which doctor attends to the highest number of patients?

Which department generates the highest revenue?

What is the average patient waiting time?

Which age group visits the hospital most frequently?

How many patients were admitted?

How many patients received treatment?

What percentage of patients were admitted?

What is the total revenue generated?

What is the average consultation fee?

Excel

## Steps followed
The following data preparation steps were completed before analysis:

Step 1: Load in data on Microsoft Excel

Step 2: Data Cleaning and validation was carried out to ensure data quality and intergrity is kept.

Step 3: Converted the Date column to the Short Date format.

Step 4: Applied the Nigerian Naira (₦) currency format to all financial columns.

Step 5: Created a Waiting Time Category column using a Nested IF formula: Fast (20 minutes or less) Moderate (21–40 minutes) Slow (Above 40 minutes)

Step 6: Created Age Group categories using a Nested IF formula.

Step 7: created pivot table

Step 8:Various visual charts like pie chart, bar chart , donut chart was used to show various insights

Step 9: visual silicers where included for department, doctor, gender and outcome
